## Install
- Node JS LTS
- Docker (Not needed for the code demo since only frontend is being presented but will be used in future)
- Python 12 (Needed for backend)

- After downloading the folder, cd into the frontend folder with `cd frontend`
- Type and enter `npm install` in the terminal to install the required Node dependencies
- Type and enter `npm run dev` in the terminal to run the website on localhost and on the network
- For some reason you may have to type and enter `npm install lucide-react` to install our icon library if it is not installed or if you see errors after runnning the website
- Ctrl+Click the localhost link to open in in the browser or copy and paste the link into your browser
